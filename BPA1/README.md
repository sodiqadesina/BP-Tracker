
# Blood Pressure Tracker (ASP.NET Core + EF Core + Identity)

A production-style, multi-user blood pressure tracker with analytics, REST API, Swagger, health checks, and structured logging.

## Features
- ASP.NET Core MVC + Identity (per-user data)
- EF Core (SQL Server) with async operations
- Service layer + ViewModels
- Dashboard (Chart.js) for trends and category breakdown
- REST API (`/api/measurements`) with filtering & pagination
- Swagger/OpenAPI in Development
- Health checks at `/health`
- Serilog console logging
